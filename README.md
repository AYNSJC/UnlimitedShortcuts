# UnlimtedShortcuts
You can use this website: https://aynsjc.github.io/UnlimitedShortcuts/ <br>
This allows you to pin as many websites as you want, all can be grouped and organised; all the code for this website is on this github repository, you can drag and drop
websites and groups and all works properly.

## Usage
If you want to use the then install the chrome extension "New Tab Redirect" form the chrome web store or enable it through this link: https://chromewebstore.google.com/detail/new-tab-redirect/icpgjfneehieebagbmdbhnlpiopdcmna?hl=en-US&utm_source=ext_sidebar
after that you can go to its settings and paste: https://aynsjc.github.io/UnlimitedShortcuts/ next time you open a tab this will redirect you to this website, here you can pin the
websites and use it.

## Disclaimer
It saves the data locally and there is no web save so, no data is backed up.
